3. I'm a returning member. How can I reconnect?
As you have no doubt noticed, there is no longer a login on this site. No forums, private messages, or member/character tracking. All communication is now done on Pandamonium's Discord server. See question #5 below for an installation walk-through. Once you join our Discord server, reach out to @Kaisa for next steps. We made our best efforts to communicate with members who were absent during the change, but obviously not everyone responded. If you held a Life Member rank when the switch occurred, we would be happy to reinstate you with the same rank. If you held a Member or lower rank, you'll have to reapply per the Charter. 5. I don't currently use Discord. Any tips?
Installation
1. Go to Discord and create an account: https://discordapp.com/register
2. Turn on 2-factor authentication to take advantage of additional features: 2FA walk-through
3. Download and install the desktop app: https://discordapp.com/
4. Click this link to the Pandamonium server: https://discord.gg/AbRtyr5

Discord Tips
Run as Administrator
Some games (like SWTOR) that must be run as administrator require Discord to also be run as administrator for push-to-talk to work correctly. Quit Discord (right click the Discord app from your task/system tray in the lower right corner and select Quit Discord - don't just close the open window, as it will continue to run in the background). Right-click the Discord shortcut and click "Run as administrator" (or select Properties, Compatibility, check "Run this program as an administrator", and then Apply). If someone can successfully use PTT while in the Discord window, but it stops working while in their game window, this is most likely the issue.

Turn Off Annoying PTT Beeps
In the lower left corner select User Settings, Notifications, and then uncheck "PTT Activate" and "PTT Deactivate".

Voice Input/Output/PTT
Under User Settings select Voice & Video and then set your Input Device, Output Device, and Input Mode (Push to Talk). This link is good to troubleshoot audio issues.

Setting an Avatar
Under User Settings select My Account, Edit, and then Change Avatar.

Turning on the Overlay
This link is a handy walk-through. Under User Settings select Overlay and Enable in-game overlay. Make sure the game you are running is being run as administrator (assuming you've now set Discord to run as administrator). Basically, they need to be the same.

Hiding Text Channels
Right-click the text channel to hide and click the Mute box. I highly recommend doing this for all channels associated with games you do not play. Then, right click the channel list box and select "Hide Muted Channels".

6. How can I stay connected to the community when not in-game?
Browser Application
You can access Discord directly from your browser: https://discordapp.com/channels/@me (be aware that Push-to-Talk does not work in the browser interface - you will need to download the desktop app to use voice chat).

Smart Phone App
The Discord smart phone app is the recommended way to stay engaged - especially if you customize the notifications so you only get pinged on channels you feel are important. It's extremely mobile friendly. Here's a quick walk-through:
  1. Install the Discord app on your smart phone: https://discordapp.com/download.
  2. From the Desktop app (only because it's easier), right click each Pandamonium text channel. You'll see some options come up, including Mute (for example, Mute #general). Check that box for those channels you do not want to get notifications on.



    You can still go in and read those channels, but you won't get pinged when someone posts there. For example, if I play FFXIV, I might leave #announcements, #applications, #introductions, and the channels in the FFXIV section unmuted.
  3. If you are a member of multiple Discord servers and you don't want to receive notifications from any servers other than Pandamonium, you can mute an entire server. Go to that server and select the server Notification Settings from the pull-down menu at the top.



    Click the Mute and Suppress options. Then click Done.
